Reflection

  1. What is the purpose of type="module" in HTML?

    The purpose is to enable importing and exporting parts of the code from different js files.

  2. Why is it useful to separate fetchUsers() into api.js?

    It is useful as we can easily see where to find fetch info to fix in case of errors or any changes.

  3. Why is it useful to separate rendering into ui.js?

    Separating code by purpose helps to navigate through code and makes it easier to locate and edit/fix.

  4. What would happen if everything stayed in one file as the project grows?

    It will become difficult to read and maintain.

  5. What is the difference between exporting and importing?

    Exporting makes the code visible to other files and makes it possible to import the code to another file in order to use it.

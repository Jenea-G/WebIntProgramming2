Reflection Questions

  1. What is inheritance?

    When we pass functionality from parent class to its child class

  2. Why do we use Animal.call(this, name) or Vehicle.call(this, brand) in child constructors?

    We do that to link parent constructor to child and thus child gets its parents properties

  3. Why do we use Object.create(...)?

    It allows us to use methods from parent.prototype

  5. What is the difference between inherited methods and child-specific methods?

    Inherited methods are defind in a parent and will be shared by all children, while child-specific methods are unavailable for other children of the same parent

  6. How does class ... extends ... relate to prototypes?

    It allows to create a parent-child link between two classes and thus transfer the parent methods to the child.
